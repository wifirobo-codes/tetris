# Tetris (Pygame)

A modern Tetris clone by WifiCodes (wifirobo)

---

## Features

- Random Tetromino generation
- Ghost piece preview
- Next piece preview
- Score, level, and line counter
- Increasing speed as you level up
- Sound effects and background music
- Pause menu
- Game Over screen
- Hard drop & soft drop
- Colored block textures

---

## Requirements

- Python 3.10+
- Pygame

Install Pygame:

```bash
pip install pygame
```

---

## Running the Game

```bash
python main.py
```

---

## Controls

| Key | Action |
|------|--------|
| ← | Move Left |
| → | Move Right |
| ↓ | Soft Drop |
| ↑ | Rotate |
| Space | Hard Drop |
| P | Pause / Resume |
| Enter | Start Game |
| R | Restart (Game Over) |
| Esc | Quit |
